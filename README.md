# Tajawal

This repository contains an AI-powered **travel assistant** For Arab speaking countries with:
-  **Recommender System** (Personalized travel suggestions)
-  **Chatbot** (Conversational LLM for travel queriesa and cultural info)
-  **Translation Model** (Multilingual support for travelers in the arab speaking countries)

##  Project Structure
- `recommender/` → Hybrid ML-based travel recommendations
- `chatbot/` → AI-powered assistant using NLP
- `translation/` → Neural machine translation module
